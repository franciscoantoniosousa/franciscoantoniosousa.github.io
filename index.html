Side Projects

🌻 Sunflower Game Engine

Sunflower is a lightweight, custom game engine built with a focus on flexibility, performance, and rendering efficiency. This project is currently in active development and serves as both a learning experience and a powerful tool for building 2D/3D applications.

🚀 Features

  • Real-time Rendering: Powered by OpenGL, enabling efficient and flexible rendering for both 2D and 3D graphics.
  • Math Utilities: GLM for vector, matrix, and transformation operations.
  • Window & Input Management: GLFW for easy window creation and input handling.
  • Image Processing: OpenCV integration for image manipulation and loading.
  • Audio Support: SFML for sound playback and audio manipulation.
  • Logging: spdlog for easy, customizable, and performant logging.

🔧 Technologies Used

  • C++: Core language for engine architecture.
  • OpenGL: Rendering backend for real-time graphics.
  • GLM: Mathematics library for graphics applications.
  • GLFW: Cross-platform windowing and input management.
  • SFML: Simple and fast multimedia library for audio support.
  • OpenCV: Image processing and manipulation.
  • spdlog: High-performance logging library.
